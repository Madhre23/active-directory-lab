# Active Directory Basics Lab (TryHackMe)

Hands-on Active Directory lab simulating a Windows domain environment (users, groups, GPO, authentication)

## Overview
This project documents a hands-on Active Directory lab completed on TryHackMe. The lab simulates a Windows domain environment and covers core concepts used in enterprise IT infrastructure, including user management, group policies, authentication, and domain architecture.

## Objectives
- Understand how Windows domains operate
- Learn how Active Directory manages users, computers, and permissions
- Explore authentication methods and security concepts
- Gain familiarity with enterprise network structures

## Technologies & Tools
- Windows Server (Domain Controller)
- Active Directory Users and Computers (ADUC)
- Group Policy Management
- TryHackMe virtual lab environment

## Lab Topics Covered
- Windows Domains
- Active Directory Fundamentals
- Managing Users in Active Directory
- Managing Computers in a Domain
- Group Policy Objects (GPO)
- Authentication Methods (Kerberos basics)
- Trees, Forests, and Trust Relationships

## Tasks Performed
- Created and managed user accounts within Active Directory
- Organized users and computers into appropriate structures
- Applied and analyzed Group Policy configurations
- Explored authentication mechanisms in a domain environment
- Reviewed domain hierarchy including trees and forests

## Key Concepts Learned
- Role of Domain Controllers in authentication and authorization
- Difference between users, groups, and Organizational Units (OUs)
- Importance of centralized management in enterprise environments
- Basics of Kerberos authentication
- How Group Policy enforces system-wide configurations
